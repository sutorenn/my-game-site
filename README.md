# Game Vault — ゲーム紹介サイト（初心者向け）

HTML と CSS だけで動く、シンプルなゲーム紹介ページです。

## ファイル構成

```
my-game-site/
├── index.html   … ページの内容（文章・構成）
├── style.css    … 見た目（色・レイアウト）
└── README.md    … この説明書
```

## 表示のしかた

1. エクスプローラーで `my-game-site` フォルダを開く
2. **`index.html` をダブルクリック**（ブラウザで開きます）

または、Cursor / VS Code の **Live Server** 拡張機能を使うと、保存のたびに自動でプレビューできます。

## 編集のヒント

| 変えたいもの | 編集するファイル |
|-------------|------------------|
| ゲームの説明文 | `index.html` の `<p class="card__desc">` など |
| 色・フォント・余白 | `style.css` の `:root` や `.card` など |
| サイト名 | `index.html` の `<title>` と `.logo` |

## 紹介しているゲーム

- **VALORANT** — タクティカル FPS
- **Minecraft** — サンドボックス
- **原神** — オープンワールド RPG

※ 各タイトルは各社の商標です。本サイトは学習用のサンプルです。
